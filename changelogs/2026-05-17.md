## Migrate từ Service Account sang OAuth2

**Files:** `main.py`, `.gitignore`

**Context/Lý do:** Google Drive API không cấp storage quota cho Service Account trên tài khoản Gmail cá nhân, dẫn đến lỗi `storageQuotaExceeded` khi upload. Chuyển sang OAuth2 để upload vào My Drive của user thật.

**Chi tiết:**
- Thay `google.oauth2.service_account` bằng OAuth2 flow dùng `InstalledAppFlow`
- Thêm `get_credentials()`: load token từ `token.json`, auto-refresh khi hết hạn, mở browser login lần đầu
- Build Drive service một lần ở `__main__` và truyền vào `handler()` thay vì tạo mới mỗi lần
- Bỏ bước share file (không cần thiết khi upload vào Drive của chính mình)
- Thêm `credentials.json`, `oauth_credentials.json`, `token.json` vào `.gitignore`

## Smart zip: auto-detect docker-compose và include_paths

**Files:** `main.py`, `config.json`

**Context/Lý do:** Mỗi folder backup cần zip các subfolder khác nhau — `be` zip uploads/extensions, `infra` zip config+data (bỏ logs), `controller` chỉ zip static. Thay vì zip cả folder, cần zip đúng phần cần thiết.

**Chi tiết:**
- Thêm `get_target_paths()`: ưu tiên `include_paths` từ config, fallback auto-detect volume mount từ `docker-compose.yml`, fallback zip cả folder
- `include_paths` hỗ trợ cả file lẫn folder (vd: `nginx/nginx.conf`)
- Thay `shutil.make_archive` bằng `zipfile` trực tiếp để handle Unicode encoding errors
- Config `infra`: zip `nginx/nginx.conf`, `nginx/lua`, `data/database`, `redis-volume` — bỏ logs, loki, promtail
- Config `controller`: chỉ zip `static/`

## Fix zip file có tên chứa surrogate characters

**Files:** `main.py`

**Context/Lý do:** Một số file trên server có tên chứa bytes invalid UTF-8 (surrogate characters) khiến `zipfile` crash khi ghi header. Cần backup được những file này thay vì bỏ qua.

**Chi tiết:**
- Thêm helper `_zip_write()`: encode tên file với `surrogateescape` rồi decode lại `latin-1` để zipfile ghi header không lỗi
- Dùng `ZipInfo.from_file()` + `writestr()` thay vì `zf.write()` để kiểm soát arcname
- Bỏ try/except skip — tất cả file đều được backup
